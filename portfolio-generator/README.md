# 🚀 Portfolio Generator

> Free AI-powered tool to create professional portfolios and CVs in minutes.

## ✨ Features

- 6 Portfolio Templates + 2 CV Templates
- AI Bio Generator (Claude API)
- ATS Score Analyzer
- Job Description Matcher
- AI Cover Letter Writer
- GitHub Stats Widget
- Dark / Light Mode
- Export: HTML, PDF, JSON, TXT
- Version History
- Shareable Link + QR Code

## 🛠️ Tech Stack

HTML5 · CSS3 · Vanilla JavaScript · Claude API

## 🚀 Live Demo

👉 [yourusername.github.io/portfolio-generator](https://yourusername.github.io/portfolio-generator)

## 🔑 Setup

1. Clone the repo
2. Open `app.js`
3. Replace `YOUR_CLAUDE_API_KEY` with your key from [console.anthropic.com](https://console.anthropic.com)
4. Open `index.html` in a browser — done!

## 📦 Deploy to GitHub Pages

1. Push to GitHub
2. Settings → Pages → Branch: main → Save
3. Your site is live in 2 minutes!
